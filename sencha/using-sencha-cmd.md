## Sencha Cmd 사용법

### Generating Views

Adding a view to your application is similar:

```cmd
$ pwd
~
$ sencha generate view -base <Base Class Name> <View Class Name>

```

### Sencha Cmd Upgrade  방법

새로운 업그레이드 버전을 체크

```cmd
sencha upgrade --check
```

마지막 버전으로 업그레이드

```cmd
$> sencha upgrade
```
